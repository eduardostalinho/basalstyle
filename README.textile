h1. BasalStyle

Conjunto básico de CSS e HTML para prototipação rápida de Sites, WebApps e Formulários.

Para entender como aplicar no documento HTML, estude os exemplos no index.html.


h2. Sobre o grid de colunas

A largura máxima é de 1080px divididos em 12 colunas.

O afastamento do texto com relação às colunas, está no "paddind" dos elementos HTMLs, como: cabeçalhos (H1,H2,..), parágrafos e listas.
